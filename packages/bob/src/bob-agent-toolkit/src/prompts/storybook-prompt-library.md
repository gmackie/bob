# Storybook Prompt Library

## Component Generator

```md
Create a React component plus Storybook stories.

Component: [X]

Include:
- all meaningful states
- error variants
- responsive variants
- accessibility variants

Use realistic mock data.
```

## State Expander

```md
Enumerate all UI states for this component including:
- happy path
- loading
- error variants
- empty
- partial data
- slow network
- invalid input
- edge UX cases

Then generate Storybook stories for all of them.
```

## UX Improver

```md
Improve this component:
- reduce cognitive load
- improve hierarchy
- clarify system feedback
- make uncertain states feel actionable

Update stories accordingly.
```

## Variant Explorer

```md
Generate 5 UX approaches to this component:
- minimal
- dense enterprise
- playful
- accessibility-first
- mobile-first

Each variant must keep full Storybook coverage.
```

## Adversarial Tester

```md
Generate mock data that will break this UI:
- extremely long text
- null fields
- malformed values
- multilingual strings
- emoji and unicode-heavy copy
- empty collections

Generate stories showing the failure points.
```
